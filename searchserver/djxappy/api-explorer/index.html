API Explorer

Try out the Xapian web service here.

Examples

Click a link to open it in the API Explorer.

/xapian/newdb?db=new-test-db
Create a new test database (do this first)
/xapian/add/new-test-d?text=this+is+a+document
Add a document to the new-test-db index.
/xapian/search/new-test-db?q=document
Run a search.
